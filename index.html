Raw Input

Voice reports, text messages, form submissions, and real-time signals flow into GridSense from multiple channels.

Crowd Safety Analyzer

When someone reports "it's getting really crowded near Stage B" or "there's pushing happening," GridSense doesn't just log it — it extracts structured intelligence: location, severity, panic probability, and recommended action.

  • → Parses voice-to-text and typed reports with NLP
  • → Classifies urgency from 1–5 with confidence scores
  • → Maps incidents to venue grid coordinates
  • → Auto-triggers evacuation or redirect protocols

Event Assistant AI

Attendees ask questions in plain language — "Where's the main stage?", "What time does Session 3 start?", "What's the shortest route to the food court?" — and receive instant, context-aware answers with venue-specific navigation.

  • → Trained on event schedules, maps, and FAQs
  • → Provides step-by-step navigation paths
  • → Considers real-time crowd density for routing
  • → Supports multi-language queries
